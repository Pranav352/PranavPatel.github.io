# Personal Portfolio Website

A responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript** to showcase projects, skills, and contact information.

This project demonstrates front-end development fundamentals including responsive layout design, interactive UI elements, and clean code organization.

---

## Features

* Responsive design for desktop and mobile
* Project showcase section
* Skills and experience section
* Contact information section
* Smooth scrolling navigation
* Clean and minimal UI

---

## Tech Stack

* HTML5
* CSS3
* JavaScript

---

## Project Structure

```text
portfolio-website/
│
├── index.html      # Main webpage
├── style.css       # Styling and layout
├── script.js       # JavaScript functionality
└── README.md       # Project documentation
```

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio-website.git
```

Navigate to the project folder:

```bash
cd portfolio-website
```

Open the project in your browser:

```bash
open index.html
```

or simply double-click `index.html`.

## Customization

You can easily customize the portfolio:

* Update **personal details** in `index.html`
* Modify **colors and layout** in `style.css`
* Add **interactivity** in `script.js`
* Add new **projects and skills**
